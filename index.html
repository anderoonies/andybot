ANDY_3000

Twitter bot that makes fun of me using Markov chaining

View on GitHub Download .zip Download .tar.gz

ANDY_3000 is a Twitter bot made to make fun of me. By using Markov text chaining and the Twitter API, ANDY_3000 publishes gibberish tweets that sound similar to things I might say.
